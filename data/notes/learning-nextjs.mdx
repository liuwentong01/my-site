## App Router Key Concepts

### Server Components

By default, all components in the App Router are Server Components. They run on the server and send HTML to the client.

### Client Components

Add `'use client'` directive at the top of the file to make a component a Client Component.

### Layouts

Layouts wrap pages and preserve state across navigation. The root layout is required.

### Loading and Error States

Use `loading.tsx` and `error.tsx` for automatic loading and error UI.
